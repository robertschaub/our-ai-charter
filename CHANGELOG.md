# Changelog

Notable changes to the charter and its drafts. Dates are ISO (YYYY-MM-DD).

## 2026-06-15
- **Repository governance** — added document-only agent and GitHub safety scaffolding adapted from FactHarbor: public/private boundary guidance, destructive-git guard references, a security policy, PR checklist, and local artifact ignores.

## 2026-06-14
- **Manifesto** — *Trustworthy AI, Accountable to People* published.
- **Protocol v0.2** — Grounding-Faithfulness & Contestability, hardened after an adversarial multi-model red-team:
  - added a **source-validity floor** (a cited source must exist and not be satire/fiction);
  - replaced "truth" with a **graded support rubric** (verbatim → paraphrase → entailment → partial → unsupported → contradicted);
  - switched to a **pre-registered, auditor-controlled** held-out sample (privacy + anti-cherry-picking);
  - corrected the **statistics** (no fixed N or threshold; pass rule on a confidence bound; clustering handled);
  - reframed outputs as an **evaluation report**, not a certificate/mark (that's a later phase);
  - **split contestability** so small vendors can comply (user-complaint for all; independent escalation only for consequential systems);
  - added **over-hedge / abstention** checks and a realistic cost.
- **Protocol v0.1** — initial draft of the evaluation protocol.

## 2026-06-13
- **Founding Accord v0.2** — public charter (values + commitments) drafted.
