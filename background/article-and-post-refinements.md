## Edit A — "How you'd know it's real": make "start light" concrete (recommended)

*Turns a hand-wave into an obviously-doable Phase 1 — the red-team's core advice was "start cheap, pilot, don't promise the pyramid."*

- **Current:** "It can start light and ride accreditation infrastructure that already exists. A direction, not a finished scheme — but the difference between a slogan and a standard."
- **Proposed:** "It can start light — a published code, a handful of independent assessors, a public registry, and the power to revoke — the way fact-checking's own trust network already runs — then grow into the full accreditation chain only at scale. A direction, not a finished scheme — but the difference between a slogan and a standard."

## Edit B — "How you'd know it's real": name what an honesty audit checks (optional)

*Pre-empts the #1 expert objection ("you can't certify truth"). Obligation #3 already gestures at it, so this is optional.*

Insert after *"…no single actor in the chain checks their own work."*:

> "And auditing honesty doesn't mean ruling on truth: it means checking whether a system's claims are actually backed by the sources it cites, whether it flags doubt, and whether it fixes errors instead of repeating them. That's measurable — 'is it true?' isn't anyone's to certify."

## Edit C — "Who could drive it": the fact-checking wedge + "convene, don't run" (recommended)

*The deep dive's sharpest finding — information-integrity makes the fact-checking world the natural methodology home — plus an honest line about a solo founder's real role.*

- **Current:** "The model: a small founding coalition convenes, the standard is hosted inside an existing body, and groups already doing the work — MLCommons, the AI Alliance, Current AI — anchor it."
- **Proposed:** "The model: a small founding coalition convenes; the standard is hosted inside an existing body; technical anchors like MLCommons and the AI Alliance supply the rigour — and because the hard part here is honesty, the people who already run trust badges for truth-telling (the fact-checking networks: IFCN, EFCSN, RSF's Journalism Trust Initiative) supply the method. The initiator convenes — no one has to build a new audit empire."
