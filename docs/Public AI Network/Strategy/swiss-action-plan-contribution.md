# Swiss AI Action Plan — contribution approach

A go/no-go layer for filing a public-interest contribution into the Swiss AI Action Plan: where it goes, what bar it must clear, and the discipline the submission itself must hold.

## The filing target

- **Where:** the Swiss AI Action Plan, **governance track** ("innovation-friendly, streamlined AI governance"), via the digitalswitzerland participation platform (`chat.digitalswitzerland.ch`, access on request). Coordinated by digitalswitzerland with BAKOM.
- **Timing:** no imminent hard deadline — structured rounds have passed and contribution is rolling through 2026. The soft window is *before the plan finalises*; verify whether a closing date has been published.
- **Framing constraint:** the official track is light-touch / voluntary-codes. A contribution must read as **additive** to that — never as a call for heavier regulation or a new institution.

## Go / no-go criteria

**File only if all three hold:**

1. **At least one substantive Swiss signal** — a real orientation call agreed or held with a credible Swiss actor (e.g. an EPFL/ETH AI-centre figure, the digitalswitzerland AI-Action-Plan team, or a neutral policy convenor), **or** an explicit "yes, bring this in" from someone inside the Action-Plan process. A like or a polite "interesting" does not count.
2. **At least one relevant international acknowledgment** — a substantive reply from a public-AI or evaluation peer/body (see the [actor & landscape map](../Evidence/actors-and-landscape.md)) that *engages the framing* — a pointer, an introduction, or real critique. An auto-reply or bare "thanks" does not count.
3. **The one-pager has passed the overclaim audit** (below).

**Hold / no-go — pause and rethink — if any:**

- A credible actor in the official process says the contribution **duplicates or conflicts** with existing work (SNAI / BAKOM / digitalswitzerland already cover it) → reframe or stand down.
- Repeated substantive feedback that the framing is **off-thesis or overclaims** → fix before filing.
- **No substantive reply** from any priority contact within the set silence window → keep building mandate; don't file into silence.
- Any sign the input reads as **"Switzerland-first," anti-industry, or as founding a new institution** → stop; the framing failed.

## Submission artifact

- **Title (for reviewers):** *"Open Evaluation and Public-AI Governance: Contributor Input for Smart AI Governance."*
- **Base:** the [Non-Paper](../Outreach/non-paper.en.md).
- **Scope:** one track (Smart AI Governance) only — no cross-track pitch; ~1 page; contributor *input*, not a formal action proposal.

**Overclaim audit (must pass before filing):**

- [ ] No certification / Trust Mark claim
- [ ] No audit-authority claim
- [ ] No FactHarbor maturity claim — at most "early open-source prototype exploring evidence-record formats"
- [ ] No implied endorsement or existing coalition (intent ≠ fact)
- [ ] Additive to "innovation-friendly governance"; not a heavier-regulation or new-institution pitch
- [ ] No private correspondence or contact data included
